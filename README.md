# Biznex

Biznex is an open-source project designed to provide full details about a company.

## Technologies

This project is built using:
- **Backend:** Java 21, Spring Boot 3.4.3
- **Database:** PostgreSQL

Frontend is not available yet.

## Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/alizoog/biznex-backend.git
   cd biznex-backend
   ```
2. **Set up the database:**
   - Install PostgreSQL and configure it in `.env` or `application.yml`.
3. **Run the project:**
   ```sh
   ./mvnw spring-boot:run
   ```
   or
   ```sh
   mvn spring-boot:run
   ```

## Usage

The backend provides API endpoints to manage and retrieve company details. API documentation will be added later.
